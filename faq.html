Adding servers for versions above 0.7

1. Join the game

2. Press the "Play" button on the screen

3. Click the "Edit" button, after "External"

4. Enter the server name in the "Server Name" field. It can be any

5. Enter the server's IP address in the "Address" field

6. Enter the server port in the "Port" field

7. Click the "Add server" button

Adding servers for versions below 0.7

This works for versions: 0.8.1, 0.7.6, 0.6.1, 0.5.0, 0.4.0, 0.3.3, 0.2.2, 0.1.3, 0.1.1

1. Download PyDroid 3 app> Click here <

2. Download the required file > Click here <

3. Open the downloaded PyDroid

4. Click on the folder icon in the upper right corner, then click "Open"

5. Find the file we downloaded (proxy.py) and open it

6. Find line 76, replace myserver.ddns.net with the ip of the server you want to add

7. If you need to change the default port (19132), find line 10 and change 19132 to your port

8. Click on the start button in the lower right corner

9. Click on the space icon next to the TAB button in the upper right corner

10. Now you can play on the server!
